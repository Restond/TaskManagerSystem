## [1.0.0] - 2025-11-10
- 项目初始化，完成基础架构
- 集成 Spring Boot & JWT 安全认证
- 实现用户注册、登录（JWT 校验）
- 实现任务的创建、更新、删除、条件查询
- 支持任务状态与优先级筛选、截止日期筛选
- 增加全局异常处理（任务不存在、任务冲突）
- 实现基于权限的路由控制
- 集成 Lombok 简化代码
- 完成基础接口文档

## [1.0.1] - 2025-11-11
### Added
- 任务状态枚举：PENDING, IN_PROGRESS, COMPLETED。
- 数据验证：任务标题不能为空且长度在1-100之间，优先级必须在1-5之间。
- 全局异常处理：新增验证异常处理，返回详细的字段错误信息。
### Changed
- 将completed字段替换为status枚举字段。
- 更新查询方法，使用status替代completed。
- 更新方法以使用枚举状态，并设置新任务的默认状态为PENDING。
- 更新状态查询端点，使用枚举类型，并添加了@Valid注解以触发验证。
